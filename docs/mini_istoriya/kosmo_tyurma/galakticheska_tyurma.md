# Галактическа Тюрьма
Игроков арестовывают
Сковывают наручниками, которые блокируют способности
Веду через большой зал по корридору
Аф-аф пообщался со змее-рыбой в джакузи в зале
Терминатор сидел в вентиляции и подсматривал за залом порталов.
