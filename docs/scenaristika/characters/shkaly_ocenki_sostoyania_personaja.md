## Базовый уровень: Физиология и Тело

| Критерий| Что измеряем | Шкала/формат|
|---|---|---|
| **Голод/Жажда** | Уровень базового дискомфорта| 0–10 или «сыт → голоден → истощение» |
| **Сон/Усталость** | Качество и количество отдыха + когнитивная усталость | «Отдохнувший → вялый → на грани срыва» |
| **Физическая боль** | Интенсивность и локализация | 0–10 + тип (острая/ноющая) |
| **Гормональный фон** | Влияние цикла, стресса, возраста | Описательно: «подъём / спад / нестабильность» |
| **Вещества** | Кофеин, алкоголь, лекарства, их отсутствие| «Стимуляция <--> норма <--> откат»|
| **Сенсорная перегрузка** | Шум, свет, толпа, тактильный дискомфорт| 0–10 |

## Уровень безопасности: Стабильность и Контроль

| Критерий | Что измеряем  | Шкала/формат |
|---|---|---|
| **Воспринимаемая угроза**  | «Мне безопасно» → «Я в опасности»  | 0–10 + тип (физическая/социальная/экзистенциальная) |
| **Контроль над ситуацией** | «Я управляю» → «Всё рушится без меня» | 0–10|
| **Ресурсная подушка**| Деньги, связи, знания, «план Б» | Описательно: «есть запас / на пределе / нет ничего» |
| **Предсказуемость среды**  | Знакомые ритуалы, расписание, правила | «Хаос ←→ Порядок»  |
| **Здоровье/Уязвимость** | Страх болезни, хронические состояния  | Описательно + уровень тревоги  |

## Социально-эмоциональный уровень: Связи и Признание

| Критерий| Что измеряем  | Шкала/формат |
|---|---|---|
| **Близость/Одиночество**  | Глубина последних контактов  | «Изоляция ←→ Поддержка»  |
| **Конфликтный фон** | Накопленные обиды, неразрешённые споры| Список + интенсивность 0–10 |
| **Социальный статус**  | Ощущение «я на своём месте»  | «Невидимка ←→ Авторитет» |
| **Эмпатический резерв**| Способность сопереживать без выгорания| 0–10|
| **Последнее значимое взаимодействие** | Краткая метка: «поддержка / предательство / равнодушие» | Текстовый тег + валентность |

## Когнитивно-эмоциональный уровень: Мышление и Чувства

| Критерий| Что измеряем | Шкала/формат  |
|---|---|---|
| **Текущий аффект**  | Преобладающая эмоция| Гнев / Страх / Радость / Печаль / Спокойствие + интенсивность 0–10 |
| **Уровень стресса** | Напряжение нервной системы| 0–10 + тип (острый/хронический)|
| **Ментальная усталость**  | Истощение внимания, памяти, воли| «Ясен ум → туман → выгорание»  |
| **Когнитивные искажения** | Склонность к катастрофизации, чёрно-белому мышлению и т.п. | Список активных искажений|
| **Внутренний диалог**  | Тон самоговорения | «Поддержка ←→ Самокритика»  |
