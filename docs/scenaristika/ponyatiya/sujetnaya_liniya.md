# {{page.title}}

## Что?:  
Отдельная история второстепенного [персонажа](/docs/scenaristika/ponyatiya/personaj.html) или же главного героя, которая занимает определенное количество [битов](/docs/scenaristika/ponyatiya/bit.html).

## Зачем?:
Развитие сюжета строится на развитии конфликтов вокруг персонажей. Действие одного героя может стать катализатором или препятствием для другого, создавая переплетающиеся сюжетные линии. Наша задача — сплести из этих пересекающихся арок единую повествовательную паутину в рамках сезона, где каждая серия будет двигать вперёд события двух или трёх таких линий.
