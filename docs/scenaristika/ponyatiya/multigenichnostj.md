## Что?:
Необходимость истории быть выраженной в анимации. Другими словами - обилие элементов, которые невозможно воссоздать в реальности или вокруг реальности. 

## Зачем?:
Нужно чтобы определить - насколько ваша история далека от реального мира.

| Вид                       | Описание                                                                        | Пример                                                                                        |
| ------------------------- | ------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| Гротеск                   | Выворачивания понятия наизнанку. Выход за рамки абсолютного значения в понятии. | Светлые волосы цвета соломы и прям являются соломой. Огненая ярость полыхает настоящим огнем. |
| Метафора                  | Показ явления или предмета через аналогии.                                      | Тяжелая голова превращается то в кирпич, то в шар для боулинга.                               |
| Одушевление               | Предметы оживают                                                                | История игрушек                                                                               |
| Очеловечивание            | Существа перенимают черты и повадки людей                                       |                                                                                               |
| Взгляд на мир людей       | Взгляд на мир людей со стороны животного или неодушевленного мира               |                                                                                               |
| Мифология                 | Отсылка к мифам и легендам                                                      |                                                                                               |
| Двойные миры              | Наличие другого мира, параллельного основному миру героя                        |                                                                                               |
| Система магии и допущений | Наличие особых способностей                                                     |                                                                                               |
