# {{page.title}}

## Разница идеи и логлайна:
	- Логлайн для зрителя, идея для сценариста.
	- Логлайн держит интригу, идея - конкретная.

## Что такое идея( для каждого 1-2 предложения):  
	- Отвечает на вопрос "кто?"(Главный герой, антагонист)
	- Отвечает на вопрос "где?"(Описываем Мир, время и нюансы)
	- Отвечает на вопрос "что делает?"(Описание конфликта)
	- Отвечает на вопрос почему он это делает(Мотивация героев и злодеев)
	- Описывает чем это закончилось. (Разрешение конфликта)
	- Описывает Посыл (Что мы хотим показать воплащая эту идею)

## Что должно быть в идее:  
	- Желание героя(Герой не может не хотеть)
	- Конфликт
	- Визуальный аттракцион
	- Мультигеничность
	- Логичность(вытекает из причины действий героя и завершается исходя из действий)
	- Посыл
	- Понятие целевой аудитории

## Что еще вкусного может быть в идее:
	- Простая жизненная ситуация
	- Трюизм(самоочевидная банальность)
	- Дедлайн
	- Выбор
	- Неожиданный поворот
	- Метафора
	- Смена целей
	- Смена путей
	- Интрига
