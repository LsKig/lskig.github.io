# {{page.title}}

## Что такое?: 
- Инструкция, подробно описывающая события истории, речь персонажей, их действия. Может содержать подробную визуализацию событий истории.

## На какие вопросы отвечает на производстве?:
- Кто сделал?, 
- Что сделал? 
- где сделал?

## Что должно быть в сценарии:
	- Персонаж(Герой) + Структура + Конфликт

## Чего не должно быть в сценарии:
- Отсутствие [урока](/docs/scenaristika/ponyatiya/posil.html).
- Отсутствие [идеи](/docs/scenaristika/ponyatiya/idea.html)
- Отсутствие [мотивации](/docs/scenaristika/ponyatiya/motivaciya.html) героя
- Недостаток знаний о том, о чем сериал.
- Много текста.
- Мало действий
- Мыслительные глаголы
