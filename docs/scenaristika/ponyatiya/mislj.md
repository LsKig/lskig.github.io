# {{page.title}}

## Что?:  
Система взаимодействия [ценностей](/docs/scenaristika/ponyatiya/cennostj.html) внутри произведения.  

## Зачем?:
Чтобы собрать наши ценности в нечто связанное. Создать связь между мотивациями сторон конфликта через ценности, которые они преследуют.

## Как?:  
Чаще всего представлена в виде конфликта ценностей
