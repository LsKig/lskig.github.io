# Бестиарий существ

## Гуль (карточка справа)

{% include creatures_card_module.html creature_key="ghoul" layout="right" %}

Гули часто встречаются на старых кладбищах. Они избегают яркого света и нападают из засады.

**Тактика боя:**
- Атакуют стаей
- Окружают жертву
- Отступают при 50% HP

**Добыча:**
- Когти гуля (алхимия)
- Личные вещи жертв

## Наккер (пример карточка слева)

{% include creatures_card_module.html creature_key="nekker" layout="left" %}

Неккеры охотятся в стаях по 3-6 особей. Предпочитают атаковать изолированных путников.

> 💡 **Совет для Мастера:** Используйте неккеров как разминку перед серьёзным боем.

---

## Голем (карточка сверху)

{% include creatures_card_module.html creature_key="golem" layout="top" %}

Големы создаются для охраны важных мест. Они не устают и не отступают.

**Особые способности:**
- 🛡️ Каменная кожа
- 💥 Землетрясение (2d8 урона)
