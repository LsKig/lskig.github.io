# Забытые боги

Репутация: 1

Нац принадлежность: Все

Описание: Культы разных забытых богов

Список культов:
	- "Та что жаждет"
	- "Тёмное облако"

Взаимодействие:

- Культ "Той что жаждет" просит разобраться с существом, которое призвал культ "Тёмное облако".
