# Дистанционный Бой
```
Атака = Лвк + Навык владения оружием + Модификатор расстояния + 1d10
```
- **Попадание:** Результат > Проверки защиты цели.
- **По неодушевленной/неподозревающей цели:** Результат > (СЛ цели + Модификатор размера).

**Арбалеты:**
- Зарядка занимает **1 действие**.

**Бомбы и ловушки:**
- **Бомбы:** Урон по зоне (все части тела). Бросок Атлетики. При провале — разброс (см. таблицу Разброс).
- **Ловушки:** Активируются при входе в зону. Урон по всем частям тела + эффект.

**Таблица разброса (1d10 направление, 1d6 метров недолёт/перелёт):**
```
      10
   7   8  9
  5  ЦЕЛЬ  6
   2   3  4
       1
```

### Дистанция и СЛ цели

| Дистанция | СЛ цели | Мод. расстояния |
| :--- | :---: | :---: |
| **В упор** (касается или < 0.5м) | 10 | +5 |
| **Близкая** (¼ дистанции оружия) | 15 | +0 |
| **Средняя** (½ дистанции оружия) | 20 | -2 |
| **Дальняя** (дистанция оружия) | 25 | -4 |
| **Экстремальная** (дистанция × 2) | 30 | -6 |
