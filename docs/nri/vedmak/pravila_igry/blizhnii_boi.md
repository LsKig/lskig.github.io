# Ближний бой
```
Атака = Реа + Навык владения оружием + Модификатор + 1d10
```

### Особые атаки ближнего боя:

| Атака | Механика |
| :--- | :--- |
| **Удар рукой/ногой** | Несмертельный урон (значения Удар рукой/ногой). Быстрая или сильная атака. |
| **Толчок ногой** | Отталкивает на (Тел / 3) метров. Наносит ½ урона. Всегда в туловище. |
| **Атака с разбега** | Полный раунд. Перемещение на Бег + Сильная атака. Штраф -3 к попаданию. Если заблокирована: встречная проверка Силы vs Силы, чтобы [сбить с ног](/docs/nri/vedmak/pravila_igry/effekty.html#knocked-off). |
| **Разоружение** | Встречная проверка Борьбы vs Уклонения/Изворотливости. Оружие отлетает на 1d6/2 м (или отбирается свободной рукой со штрафом -3). |
| **Захват** | Проверка, чтобы схватить. Цель не может отойти, штраф -2 к физ. действиям. Освобождение: встречная Уклонение/Изворотливость vs Борьба. |
| **Обездвиживание** | В захвате. Проверка, чтобы иммобилизовать. Цель не может двигаться/действовать до освобождения. |
| **Душение** | В захвате. Цель [задыхается](/docs/nri/vedmak/pravila_igry/effekty.html#suffocating) до освобождения. |
| **Бросок** | В захвате. Проверка, чтобы бросить. Цель [сбита с ног](/docs/nri/vedmak/pravila_igry/effekty.html#knocked-off), урон = Удар рукой. Испытание [Устойчивости](/docs/nri/vedmak/pravila_igry/bazovyi_boi.html#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) со штрафом -1. |
| **Подсечка** | Удар по ногам. При успехе цель [падает плашмя](/docs/nri/vedmak/pravila_igry/effekty.html#knocked-off)). |
| **Удар эфесом** | Несмертельный урон (½ от обычного). |
| **Разоружение (атакой)** | Прицельный удар. Оружие отлетает на 1d6 м (см. Разброс). |
| **Финт** | Проверка Обмана вместо 1-й быстрой атаки. Встречная Внимания цели vs Обмана. При успехе 2-я атака с бонусом +3. |

**Парное оружие:**
- Две проверки атаки, обе со штрафом **-3**.
- Противник должен иметь парное оружие (или оружие + щит), чтобы блокировать/парировать обе атаки. Иначе должен уклоняться/менять позицию.
- Можно стрелять из двух ручных арбалетов.
- Заменяет быструю или сильную атаку.

**Атака щитом:**
- Проверка атаки Ближнего боя.
- Урон = Значение Удар рукой (смертельный).
- **Средние щиты:** Урон как если бы Удар рукой был на **2 уровня выше**.
- **Тяжёлые щиты:** Урон как если бы Удар рукой был на **4 уровня выше**.

### Линия видимости и Конус зрения
- **Конус зрения:** Пространство от плеча до плеча.
- **За пределами конуса:** Штраф **-3 к атаке**, цель получает **+3 к атаке** против вас.
- **Шлемы с ограничением зрения:** Конус сужается, отменяет бонусы ведьмаков к Вниманию.
