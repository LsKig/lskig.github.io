# Эффекты
Неприятные состояния, накладываемые оружием, магией и т.д.

| Название          | Эффект                                                                                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <span id="burning"></span> **Горение**    | 5 урона по каждой части тела в ход. Броня поглощает, но огонь наносит 1 урон броне/оружию. <br>**Тушение**: облить водой или упасть/покататься (1 ход). |
| <span id="disorienting"></span> **Дезориентация**{: #disorienting}  | Не может действовать. <br>**Попасть по нему:** проверка со СЛ 10. <br>**Снятие:** испытание [Устойчивости](/docs/nri/vedmak/pravila_igry/bazovyi_boi.html#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) (1 ход) или получение урона.                   |
|  <span id="poison"></span> **Отравление**    | 3 урона в ход (не снижается бронёй). <br>**Снятие**: проверка [Устойчивости](/docs/nri/vedmak/pravila_igry/bazovyi_boi.html#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) со СЛ 15 (1 действие).                                                       |
|  <span id="bleeding"></span> **Кровотечение**  | 2 урона в ход. <br>**Снятие**: исцеляющее заклинание или Первая помощь со СЛ 15 (1 действие).                                                           |
|  <span id="freezing"></span> **Замораживание** | Штраф -3 к Скор, -1 к Реа. <br>**Снятие**: проверка Силы со СЛ 16 (1 действие).                                                                         |
|  <span id="stunning"></span> **Ошеломление**   | Штраф -2 к атаке и защите. <br>**Снятие:** Проходит к началу следующего хода.                                                                           |
| <span id="intoxicating"></span> **Опьянение**     | Штраф -2 к Реа, Лвк, Инт; -3 в Словесной дуэли. <br>25% шанс забыть события.<br>**Снятие**: Сон или слезы жен.                                          |
| <span id="hallucinating"></span> **Галлюцинации**  | Видит нереальные вещи. Отличие от реальности: проверка Дедукции со СЛ 15.<br>**Снятие:** ?                                                              |
| <span id="nausea"></span> **Тошнота**       | Каждые 3 раунда бросок 1d10 < Тел. При провале -  рвота/позывы<br>**Снятие:** проходит в конце раунда.                                                  |
| <span id="suffocating"></span> **Удушье**        | 3 урона в раунд (не снижается бронёй). <br>**Снятие**: восстановление доступа к воздуху.                                                                |
| <span id="blinding"></span> **Слепота**       | Штраф -3 к атакам и защите, -5 к проверкам Внимания (зрение). **Снятие**: 1 ход на восстановление зрения.                                               |
| <span id="unconscious"></span>**Без сознания**  | См. [Дезориентация](#disorienting) + [Сбит с ног](#knocked-off)                                                                                                                          |
| <span id="knocked-off"></span>**Сбит с ног**    | -2 к проверкам атаки и защиты<br>**Снятие**: Половина движения на то чтобы подняться                                                                    |
| <span id="imbolized"></span>**Обездвижен**    | Не может двигаться/действовать до освобождения. <br>**Снятие**: Освободиться(для захвата встречная Уклонение/Изворотливость vs Борьба)
