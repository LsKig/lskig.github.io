# Критческие Провалы

 (1d10 + разница атаки и защиты)

| Тип проверки | Результат |
|--------------|-----------|
| **Атака ближнего боя** | 1-5: нет последствий. 6: ошеломлен. 7: оружие застряло (1 раунд). 8: оружие повреждено (1d10 урона надежности). 9: ранил себя. >9: ранил союзника. |
| **Защита оружием** | 1-5: нет последствий. 6: доп. 1d6 урона надежности. 7: оружие выбито (1d6 м). 8: сбит с ног (испытание Устойчивости). 9: доп. 2d6 урона надежности. >9: рикошет по себе. |
| **Дистанционная атака** | 1-5: нет последствий. 6-7: снаряд сломался. 8-9: тетива слетела/арбалет заклинил (1 раунд). >9: рикошет по союзнику. |
| **Без оружия** | 1-5: нет последствий. 6: ошеломлен. 7: сбит с ног. 8: сбит с ног + испытание Устойчивости. 9: удар головой (1d6 несмерт. урон + испытание). >9: удар головой (1d6 смерт. урон + испытание). |
