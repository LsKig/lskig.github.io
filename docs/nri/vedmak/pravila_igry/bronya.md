# Броня
ПБ -Пункты брони. 
### Многослойная броня
Максимум 3 части брони (1 тяжелая, 1 средняя).
**Расчет общей ПБ:**
1. Вычесть ПБ более легкой брони из ПБ более тяжелой.
2. Найти разницу в таблице "Дополнительная броня".
3. Прибавить бонус к ПБ тяжелой брони.
4. Повторить для следующего слоя.

### Таблица дополнительной брони:

| Разность ПБ | Бонус к броне |
|-------------|---------------|
| 0–4 | +5 |
| 5–8 | +4 |
| 9–14 | +3 |
| 15–20 | +2 |

**Скованность движений (СД):** Сумма СД всех слоев + 1 за каждый слой средней брони + 2 за каждый слой тяжелой.

### Укрытия
Атака должна пробить ПБ укрытия. Оставшийся урон идет по броне/персонажу.

| Укрытие | ПБ |
|---------|----|
| Каменная стена / Толстое дерево | 30 |
| Кирпичная стена | 25 |
| Стальная дверь | 20 |
| Тяжёлая деревянная дверь | 15 |
| Деревянная стена / Повозка / Бочка | 10 |
| Соломенная крыша / Кустарник | 7 |
| Палатка | 5 |

### Живой щит
- Проверка Борьбы/Силы против Силы/Уклонения цели.
- При успехе цель становится укрытием для туловища.
- Урон сначала идет по цели (Тел + ПБ туловища), остаток — по туловищу владельца щита.
