# Скакуны И Транспорт
### Проверка управления
```
1d10 + Верховая езда + модификатор управления против СЛ ситуации
```

**Модификаторы управления:**

| Ситуация | Мод. |
|----------|-------|
| Спокойная езда | +0 |
| Быстрый галоп | +2 |
| Препятствие | +4 |
| Бой верхом | +3 |
| Паника скакуна | +5 |

### Бой верхом
- Атака: штраф -2, если не используется кавалерийское седло.
- Таран: урон = 1d6 + Скор скакуна, цель проходит тест Уст или падает.
- Потеря управления: при провале проверки — падение, 1d6 урона.
