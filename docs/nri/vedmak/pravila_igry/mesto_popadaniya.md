# Место Попадания
### Гуманоиды
Если не целились в конкретную часть, бросьте 1d10.

| Часть тела  | Бросок | Штраф к попаданию | Множитель урона |
| ----------- | ------ | ----------------- | --------------- |
| Голова      | 1      | -6                | ×3              |
| Туловище    | 2–4    | -1                | ×1              |
| Правая рука | 5      | -3                | ×1/2            |
| Левая рука  | 6      | -3                | ×1/2            |
| Правая нога | 7–8    | -2                | ×1/2            |
| Левая нога  | 9–10   | -2                | ×1/2            |

### Чудовища
Бросьте 1d10 (или по таблице для конкретного монстра).

| Часть тела        | Бросок | Штраф к попаданию | Множитель урона |
| ----------------- | ------ | ----------------- | --------------- |
| Голова            | 1      | -6                | ×3              |
| Туловище          | 2–5    | -1                | ×1              |
| Правая конечность | 6–7    | -3                | ×1/2            |
| Левая конечность  | 8–9    | -3                | ×1/2            |
| Хвост или крыло   | 10     | -2                | ×1/2            |
