# Испытание Против Смерти
Когда ПЗ < 0:

- Все параметры снижены до 1/3.
- Бросок 1d10 <= Уст (до штрафов).
- **Провал:** Смерть.
- **Успех:** Выживание в этот раунд.
- **Накопление:** Каждое новое испытание со штрафом -1. При получении нового урона — дополнительное испытание со штрафом -1.
