# Магическая Атака
### Магическая атака
```
Атака = Воля + Сотворение заклинаний + 1d10
```

### Энергия и Выносливость (Вын)
- **Энергия:** Лимит магической энергии за 1 раунд без вреда.
- **Правило:** Сумма затрат Вын всех заклинаний за раунд ≤ Энергия.
- **Выносливость:** Общий пул. Затраты вычитаются из Вын.
- **0 Вын:** Персонаж **Дезориентирован**. Требуется испытание [Устойчивости](/docs/nri/vedmak/pravila_igry/bazovyi_boi#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) и отдых до 20 Вын.

### Перегрузка и провал
- **Перегрузка:** Затраты > Энергия.
    - Стоимость: **5 ПЗ** за каждый пункт Вын сверх Энергии.
    - Эффект: Стихийный эффект провала (см. таблицу).
- **Критический провал (Магический):** Выпадение 1 на d10 (перебрасывать при 1).
    - **Последствия (1d10):**
        - **1-6:** Магия трещит, урон = величине провала (1 за 1 пункт). Заклинание срабатывает.
        - **7-9:** Заклинание не срабатывает. Внутренний ожог. Стихийный эффект критического провала.
        - **>9:** Взрыв. Стихийный эффект. Фокусирующий предмет взрывается (1d10 урона, радиус 2м).

### Стихийный эффект критического провала

| Элемент | Эффект |
| :--- | :--- |
| **Смешанные** | Урон (1 за 1 пункт провала) + 1 случайный эффект ниже. |
| **Земля** | Урон (1 за 1 пункт провала). Персонаж **Дезориентирован**. |
| **Воздух** | Урон (1 за 1 пункт провала). Отбрасывает на 2 м назад. |
| **Огонь** | Урон (1 за 1 пункт провала). Персонаж **Горит**. |
| **Вода** | Урон (1 за 1 пункт провала). Персонаж **Заморожен**. |

### Двимерит
- **Свойства:** Антимагический металл.
- **Эффекты:**
    - Касание: Энергия = 0.
    - В радиусе 5м: -1 Энергии за единицу металла.
    - Касание требует проверки [Стойкости](/docs/nri/vedmak/pravila_igry/bazovyi_boi#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) (СЛ 16) каждые 30 минут.
- **Таблица эффектов двимерита (Результат проверки Стойкости):**
    - **≥18:** Дискомфорт (чешется кожа).
    - **≥16:** Чешется, подташнивает.
    - **≥14:** Спазмы. Проверка [Стойкости](/docs/nri/vedmak/pravila_igry/bazovyi_boi#%D0%B8%D1%81%D0%BF%D1%8B%D1%82%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D1%81%D1%82%D0%BE%D0%B9%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D0%B8%D1%81%D1%82%D0%BE%D0%B9%D0%BA%D0%BE%D1%81%D1%82%D0%B8) (СЛ 15) каждые 1d6 ходов или **Ошеломление**.
    - **≥12:** Кожа горит, тошнота.
    - **≥10:** Невозможно сосредоточиться. Испытание Устойчивости каждый раунд.
    - **<10:** Магия возжигается в теле. **1d6 урона каждый раунд** контакта.

### Могущественные маги (Требования к сотворению)

| Уровень навыка | Требования |
|:---:|:---:|
| **1-6** | Слова + пассы руками. |
| **7-8** | Только пассы руками. |
| **9-10** | Небольшие пассы любой частью тела. |

### Фокусирующий предмет
- Снижает затраты Вын (не ниже 1).
- Нужно держать в руках.
- Только 1 предмет одновременно.
