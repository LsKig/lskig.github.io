# Защита
Все действия защиты эффективны против дистанционных и ближних атак (если не указано иное).

| Действие              | Механика                                                                                                                                                                                                                                                      |
| :-------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Уклонение**         | Встречная проверка Уклонения/Изворотливости vs Атаки противника.                                                                                                                                                                                              |
| **Изменение позиции** | Встречная проверка Атлетики vs Атаки противника. Уходите от атаки + перемещение на ½ Скор в любом направлении.                                                                                                                                                |
| **Блокирование**      | Проверка навыка владения оружием (или Ближнего боя для щита/живого щита). Отражающий предмет получает **1 урон**. **Дистанционные атаки блокируются только щитом.** Блок телом/рукой: проверка Борьбы, немедленный урон по конечности (применяется ПБ брони). |
| **Парирование**       | Штраф **-3** к проверке оружия/Ближнего боя/Борьбы. Отменяет атаку, оружие не повреждается, противник **Ошеломлён**. Нельзя парировать лук/арбалет. Метательное оружие со штрафом **-5**.                                                                     |

- **Постепенное пробивание:** Атака, пробившая броню, снижает её Прочность (ПБ) на 1. При ПБ=0 броня сломана.
- **Щиты/Оружие:** Блокирование снижает Надёжность предмета на 1. При Надёжности=0 предмет бесполезен.
- **Живой щит:**
    - Проверка Борьбы/Силы vs Силы/Уклонения «щита».
    - «Щит» получает урон первым (Тел + ПБ брони).
    - Урон, пробивший «щит», попадает в туловище владельца.
