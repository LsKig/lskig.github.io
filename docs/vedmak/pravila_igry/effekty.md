# Эффекты
Неприятные состояния, накладываемые оружием, магией и т.д.

| Название          | Эффект                                                                                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Горение**       | 5 урона по каждой части тела в ход. Броня поглощает, но огонь наносит 1 урон броне/оружию. <br>**Тушение**: облить водой или упасть/покататься (1 ход). |
| **Дезориентация** | Не может действовать. <br>**Попасть по нему:** проверка со СЛ 10. <br>**Снятие:** испытание Устойчивости (1 ход) или получение урона.                   |
| **Отравление**    | 3 урона в ход (не снижается бронёй). <br>**Снятие**: проверка Устойчивости со СЛ 15 (1 действие).                                                       |
| **Кровотечение**  | 2 урона в ход. <br>**Снятие**: исцеляющее заклинание или Первая помощь со СЛ 15 (1 действие).                                                           |
| **Замораживание** | Штраф -3 к Скор, -1 к Реа. <br>**Снятие**: проверка Силы со СЛ 16 (1 действие).                                                                         |
| **Ошеломление**   | Штраф -2 к атаке и защите. <br>**Снятие:** Проходит к началу следующего хода.                                                                           |
| **Опьянение**     | Штраф -2 к Реа, Лвк, Инт; -3 в Словесной дуэли. <br>25% шанс забыть события.<br>**Снятие**: Сон или слезы жен.                                          |
| **Галлюцинации**  | Видит нереальные вещи. Отличие от реальности: проверка Дедукции со СЛ 15.<br>**Снятие:** ?                                                              |
| **Тошнота**       | Каждые 3 раунда бросок 1d10 < Тел. При провале -  рвота/позывы<br>**Снятие:** проходит в конце раунда.                                                  |
| **Удушье**        | 3 урона в раунд (не снижается бронёй). <br>**Снятие**: восстановление доступа к воздуху.                                                                |
| **Слепота**       | Штраф -3 к атакам и защите, -5 к проверкам Внимания (зрение). **Снятие**: 1 ход на восстановление зрения.                                               |
| **Без сознания**  | См. Дезориентация + Сбит с ног                                                                                                                          |
| **Сбит с ног**    | -2 к проверкам атаки и защиты<br>**Снятие**: Половина движения на то чтобы подняться                                                                    |
