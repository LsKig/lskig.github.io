# Модификаторы Оружия И Брони

# Модификаторы оружия

| Название | Механика |
|----------|----------|
| **Незаметное** | +2 к проверкам Скрытности при попытке спрятать оружие |
| **Кровопускающее (x)** | При нанесении урона: x% шанс вызвать кровотечение у цели |
| **Пробивающее броню** | Игнорирует сопротивление урону брони. Улучшенное пробивающее также игнорирует ½ прочности брони |
| **Дезориентирующее (x)** | При ударе по туловищу или голове: цель проходит испытание Устойчивости со штрафом -x |
| **Метеоритное** | Наносит полный урон чудовищам, уязвимым к метеоритной стали. +5 к надёжности |
| **Длинное** | Можно атаковать противников в пределах 2 метров в ближнем бою |
| **Фокусирующее (x)** | При сотворении магии: вычтите x из стоимости заклинания в Вын |
| **Улучшенное фокусирующее** | При сотворении заклинаний: СЛ проверок против ваших заклинаний считается на 2 выше |
| **Захватное** | Можно использовать для захвата и подсечки противника в пределах дистанции оружия |
| **Медленно перезаряжающееся** | Для перезарядки требуется 1 действие |
| **Несмертельное** | Можно использовать для нанесения несмертельного урона без штрафов |
| **Сбалансированное** | При критическом ранении: бросьте 2d6+2 для определения места. Если атака прицельная: бросьте 1d6+1 вместо 1d6 для определения серьёзности |
| **Разрушающее** | При попадании: наносит 1d6/2 урона прочности брони |
| **Рукопашное** | Использует навык Борьба. Урон прибавляется к урону от атаки без оружия |

## Характеристики оружия

| Параметр | Описание |
|----------|----------|
| **Тип** | Режущий (Р), Колющий (К), Дробящий (Д), Стихийный (С) |
| **Точность (Т)** | Бонус к проверке атаки |
| **Доступность (Дос)** | Повсеместное (П), Обычное (О), Редкое (Р), Уникальное (У) |
| **Урон (У)** | Кости урона при попадании |
| **Надёжность (Н)** | Количество раз, которое можно блокировать удар до поломки |
| **Хват (Х)** | Количество свободных рук для держания оружия. Двуручное в одной руке: штраф -3 |
| **Дистанция (Дис)** | Дальняя дистанция атаки. «Нет» = цель должна быть рядом |
| **Скрытность (Скр)** | Маленькое (М) — карман, Небольшое (Н) — под курткой, Крупное (К) — под плащом, Н/С — нельзя спрятать |
| **Усиление (Ус)** | Количество рун, которые можно нанести на оружие |
| **Вес** | Вес в килограммах |
| **Цена** | Цена в реданских кронах |

### Правило сопротивления кровотечению:**

Если у брони есть сопротивление кровотечению, урон от кровотечения уменьшается вдвое (с округлением вниз).
*Пример:* Вместо 2 урона за раунд персонаж получает 1 урон.

## Правила применения

*   Усиления продаются цельным набором.
*   Набор можно применить ко всем частям брони, надетым на персонажа.
*   **Разделение:** По желанию можно использовать часть набора и усилить только одну деталь брони (защищающую одну область тела). Остальные усиления из набора сохраняются для использования позднее.

### Прикрепление усиления:
*   **Действие:** Действие полного хода.
*   **Требования:** Наличие инструментов ремесленника.
*   **Проверка:** Изготовление со **СЛ 14**.
    *   *Успех:* Бонусы усиления и его вес добавляются к броне.
    *   *Провал:* Усилить броню не получается, необходимо попытаться снова.
*   **Прочность:** Усиление нельзя снять путём повреждения или поломки брони. Уничтожение брони уничтожает и усиление.

**Снятие усиления:**
*   **Проверка:** Изготовление со **СЛ 15**.
    *   *Успех:* Усиление снимается и может быть прикреплено к другому доспеху.
    *   *Провал:* Усиление остаётся на месте.

### Доболнительные модификаторы оружия и брони

- **Сопротивление урону (СУ):** Уменьшает урон определенного типа (режущий, колющий, дробящий, стихийный) **вдвое** после вычета ПБ.
- **Пробивание брони:** Игнорирует СУ.
- **Улучшенное пробивание:** Игнорирует СУ и половину ПБ.
