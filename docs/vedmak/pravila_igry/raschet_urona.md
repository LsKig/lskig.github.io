## Бомбы и ловушки

- **Бомбы:** Зонированный урон. Дистанция = Тел × 4.
- **Ловушки:** Активация при входе в зону. Урон по всем частям тела.
- **При провале броска (бомба):** Таблица разброса (1d10 направление, 1d6 метров недолёт/перелёт).
