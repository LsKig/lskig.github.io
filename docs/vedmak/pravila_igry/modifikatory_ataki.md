# Модификаторы Атаки
#### Модификаторы атаки
| Ситуация | Модификатор |
|----------|-------------|
| Засада (первый раунд) | +5 |
| Быстрое выхватывание | -3 |
| Прицеливание (за раунд) | +1 |
| Рикошет | -5 |
| Ослепление светом/пылью | -3 |
| Цель активно уклоняется | -2 |
| Цель движется (Реа > 10) | -3 |
| Цель обездвижена | +4 |
| Цель вырисовывается на фоне | +2 |

### Модификаторы размера (для атак по неодушевленным/неподозревающим)
| Размер | Модификатор |
| :--- | :---: |
| Небольшой (кошка, накер) | +2 |
| Средний (человек) | +0 |
| Крупный (тролль, лошадь) | -2 |
| Огромный (бес) | -4 |

### Освещение
| Освещение | Эффект |
| :--- | :--- |
| **Яркий свет** (солнце в пустыне/снегу) | -3 к Вниманию, -3 к атаке и защите (если смотреть против солнца). |
| **Дневной свет** | Нет штрафов. |
| **Слабый свет** (лунный) | -2 к Вниманию. |
| **Темнота** (новолуние, пещера) | -4 к Вниманию, -2 к атаке и защите. |

- **Бой в воде:**
    - Скорость атаки ближним боем = 1.
    - -2 к атаке, блокированию, парированию. -3 к Вниманию.
    - Уклонение/Изменение позиции = Атлетика.
    - Метательное оружие не работает (кроме копья).
    - Лук/Арбалет: Дистанция / 4, Урон / 2, -2 к атаке.
    - **Перемещение в воде:** По 3 осям, расстояние = ½ Прыжка.
