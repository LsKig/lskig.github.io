# Базовый Бой
- Длительность **Раунд:** 3 секунды.
- **Эфирное время:** Отслеживание времени вне боя.

### Инициатива
```
Инициатива = 1d10 + Реа
```
- **Порядок:** От большего к меньшему.
- **Ничья:** Запомнить значение, перебросить 1d10 для определения порядка.
- **Групповая:** Лидер бросает 1d10 + Реа. Результат прибавляется к Реа каждого члена группы.

### Действия
За один раунд: **Перемещение** (Скор метров) + **1 Действие**.

| Тип действия       | Примеры                                                                                                                                                                                                                |
| :----------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Стандартное**    | Атака, Сотворение магии, Использование навыка, Взять/выхватить предмет.                                                                                                                                                |
| **Полный ход**     | **Бег** (Скор × 3), **Активное уклонение** (-2 к атакам по вам в ближнем бою), **Прицеливание** (+1 к атаке, макс. +3), **Отдых** (восстановление Вын = Отдых).                                                        |
| **Дополнительное** | **Атака:** Потратить 3 Вын (штраф -3). **Защита:** Каждое доп. действие защиты (после первого) стоит 1 Вын (не требуется при Активном уклонении). Сумма энергии атакующих заклинаний меньше Энергии, иначе перегрузка. |
### Специальные маневры
- **Засада:** Проверка Скрытности против Внимания цели. При успехе: +5 к атаке против тех, кто не заметил. Инициатива проверяется после первой атаки.
- **Объединение сил:** Штраф **-1 к защите** за каждого противника в ближнем бою (сверх первого).
- **Отложенное действие:** Действовать позже в раунде (до или после персонажа с меньшей инициативой). Нельзя отложить на следующий раунд.
- **Быстрое выхватывание:** +3 к инициативе, -3 к атаке. Обязательная атака. Можно выхватить оружие в процессе.
- **Скорость атаки (СА):** Параметр для чудовищ/NPC. Показывает количество ударов за раунд. Не могут совершать быстрые/сильные атаки.

## Критические успехи и провалы (Общие)

- **Критический успех (10):** Бросить ещё 1d10, прибавить. Если снова 10 — повторить.
- **Критический провал (1):** Бросить ещё 1d10, вычесть из основы. Если снова 10 — повторить и вычесть. Основа не ниже 0. 
	- **Последствия критического провала в бою:** Зависят от вычтенного значения (см. таблицы последствий например, ошеломление, поломка оружия, ранение союзника).

### Влияние среды
- **Экстремальная жара:** Выносливость -1/3. В средней/тяжёлой броне Выносливость -1/2.
- **Болотистая местность/Заросли:** -2 к Уклонению/Изворотливости и Атлетике.
- **Лёд и снег:** +3 к Выживанию (свежие следы), -3 (старые следы). Проверка Атлетики (СЛ 14) после бега/атаки, чтобы не упасть. Голый/легкая одежда: выживание часов = Уст, затем состояние "При смерти".

### Типы урона
- **Смертельный:** Снижает ПЗ (Пункты Здоровья).
- **Несмертельный:** Снижает Вын (Выносливость). При Вын=0 персонаж дезориентирован.

### Испытание устойчивости(Стойкости)
Бросок 1d10 < порог устойчивости
При провале - дезориентация если не указано иное.
